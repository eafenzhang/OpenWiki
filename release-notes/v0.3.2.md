## ✨ What's New

- The app is now signed and notarized by Apple — double-clicking the DMG no longer shows the "OpenWiki cannot be opened because Apple cannot check it for malicious software" warning. Just install and run.

## Installation (macOS)

1. Download the `.dmg` file that matches your Mac:
   - **Apple Silicon (M1/M2/M3/M4)**: `OpenWiki_0.3.2_aarch64.dmg`
   - **Intel Mac**: `OpenWiki_0.3.2_x64.dmg`
2. Double-click the `.dmg` and drag **OpenWiki** into the **Applications** folder.
3. Double-click OpenWiki to launch — no first-run security prompt anymore.
4. Go to **Settings → AI** to configure your AI provider (Claude / OpenAI / Gemini).
